### Use ConfigMaps and Secrets to configure applications

#### Configmap and Secret Env variable

##### As individual values
```yaml
 spec:
   containers:
   - name: demo-container
     image: busybox
     command: ["/bin/sh", "-c", "env && sleep 3600"]
     env:
       - name: THEME
         valueFrom:
           configMapKeyRef:
             name: app-config-declarative
             key: ui.theme
       - name: DB_PASSWORD
         valueFrom:
           secretKeyRef:
             name: db-credentials
             key: password
```
##### Complete configmap as envFrom
```yaml
 spec:
   containers:
   - name: demo-container
     image: busybox
     command: ["/bin/sh", "-c", "env && sleep 3600"]
     envFrom:
     - configMapRef:
        name: app-config
```
##### Individual values from secret keys on env
```yaml
 spec:
   containers:
   - name: demo-container
     image: busybox
     command: ["/bin/sh", "-c", "env && sleep 3600"]
     env:
       - name: DB_PASSWORD
         valueFrom:
           secretKeyRef:
             name: db-credentials
             key: password
```
##### Complete secret as envFrom
```yaml
 spec:
   containers:
   - name: demo-container
     image: busybox
     command: ["/bin/sh", "-c", "env && sleep 3600"]
     envFrom:
     - secretRef:
        name: app-secret
```
