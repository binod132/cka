# 3.1.1 Mastering Deployments

K8s doc: https://kubernetes.io/docs/concepts/workloads/controllers/deployment/
### Performing a Rolling Update
##### Create deployment menifest

```sh 
k create deployment nginx-deployment --image nginx --dry-run -o yaml > deployment.yaml
```
```sh
kubectl apply -f deployment.yaml
```

##### Update the Container Image to trigger the rolling update
```sh
kubectl set image deployment/nginx-deployment nginx=nginx:1.25.0
```
##### Perform rolling update to restart all pods
```sh
kubectl rollout restart deployment nginx-deployment
```

##### Observe the rollout
```sh
kubectl rollout status deployment/nginx-deployment
kubectl get pods -l app=nginx-deployment -w
```
##### Executing and Verifying Rollbacks
###### View Revision History:
```sh
kubectl rollout history deployment/nginx-deployment
```
###### Roll back to the previous version:
```sh
kubectl rollout undo deployment/nginx-deployment
```
###### Roll back to a specific revision (e.g., revision 1):
```sh
kubectl rollout undo deployment/nginx-deployment --to-revision=1
```

### Use ConfigMaps and Secrets to configure applications

#### Configmap and Secret Env variable

##### As individual values
```yaml
 spec:
   containers:
   - name: demo-container
     image: busybox
     command: ["/bin/sh", "-c", "env && sleep 3600"]
     env:
       - name: THEME
         valueFrom:
           configMapKeyRef:
             name: app-config-declarative
             key: ui.theme
       - name: DB_PASSWORD
         valueFrom:
           secretKeyRef:
             name: db-credentials
             key: password
```
##### Complete configmap as envFrom
```yaml
 spec:
   containers:
   - name: demo-container
     image: busybox
     command: ["/bin/sh", "-c", "env && sleep 3600"]
     envFrom:
     - configMapRef:
        name: app-config
```
##### Individual values from secret keys on env
```yaml
 spec:
   containers:
   - name: demo-container
     image: busybox
     command: ["/bin/sh", "-c", "env && sleep 3600"]
     env:
       - name: DB_PASSWORD
         valueFrom:
           secretKeyRef:
             name: db-credentials
             key: password
```
##### Complete secret as envFrom
```yaml
 spec:
   containers:
   - name: demo-container
     image: busybox
     command: ["/bin/sh", "-c", "env && sleep 3600"]
     envFrom:
     - secretRef:
        name: app-secret
```
