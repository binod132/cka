### Kustomization
K8s Documentation: https://kubernetes.io/docs/tasks/manage-kubernetes-objects/kustomization/
#### Demo: Customizing a Deployment with Kustomize
##### Create base manifest (my-app/base/deployment.yaml)
```yaml
apiVersion: apps/v1
 kind: Deployment
 metadata:
   name: my-app
 spec:
   replicas: 1
   selector:
     matchLabels:
       app: my-app
   template:
     metadata:
       labels:
         app: my-app
     spec:
       containers:
       - name: nginx
         image: nginx:1.25.0
```
##### Create base Kustomization file (my-app/base/kustomization.yaml):
```yaml
resources:
- deployment.yaml
```

#### Create production overlay and patch
##### my-app/overlays/production/patch.yaml

```yaml
apiVersion: apps/v1
 kind: Deployment
 metadata:
   name: my-app
 spec:
   replicas: 3
```
##### Create my-app/overlays/production/kustomization.yaml
```yaml
bases:
-../../base
patches:
- path: patch.yaml
```
